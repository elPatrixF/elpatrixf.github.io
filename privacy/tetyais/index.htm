Privacy policy

This application is a videogame. It does not collect user data of any kind

About Location use.

This game uses the Bluetooth File Transfer extension, which due to the way it's implemented requires the following permissions. android.permission.ACCESS_FINE_LOCATION android.permission.ACCESS_COARSE_LOCATION
Despite this, this game does NOT access the GPS for your location, it does not access your location in the background, and the game can be fully played without geolocation.
The bluetooth extension is used for local multiplayer functionality. This is the Bluetooth Object in question, https://clickstore.clickteam.com/plugins/extensions/bluetooth_object

Data collected by the app

I have not programmed this app to collect any personally identifiable information. It can be safely used without internet connection.

This app also does not offer any kind of monetization feature, nor accepts any payments in any form. This app shall be distributed for free, with no analytics of any kind implemented on it.
